Improved the chatbot with LLM API integration - Llama2, incorporating the trained model for real-time medical classification. Developed the chatbot for symptom analysis, data queries using Tokenization, and automated appointment scheduling.

2. Lip Reading: TensorFlow & CNN (Click here)

This lip reading project has equipped me with valuable insights into utilizing Convolutional Neural Networks (CNNs) and advanced deep learning techniques like Bidirectional LSTMs and CTC loss for decoding spoken words from lip movements. The project covered key aspects, including data preprocessing with TensorFlow, building an efficient data pipeline, and training a model to accurately predict and translate lip movements into text. The integration of CNNs facilitated robust feature extraction, while Bidirectional LSTMs enhanced the model's understanding of temporal dependencies in sequential data. The use of the CTC loss function showcased its effectiveness in handling sequence-to-sequence tasks, making this project a practical exploration of lip reading using cutting-edge deep learning approaches.

It also highlighted the significance of feature selection in machine learning and the potential impact of attribute selection techniques on model performance. The project's hands-on approach allowed me to gain valuable insights into the machine learning process and its applications in various domains.
